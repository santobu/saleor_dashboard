# Saleor Dashboard

### Installing

Clone the repository:

```
$ git clone https://github.com/santobu/saleor_dashboard.git
```

Enter the project directory:

```
$ cd saleor-dashboard
$ git checkout main
```

Install NPM dependencies:

```
$ npm i
$ npm start
```
